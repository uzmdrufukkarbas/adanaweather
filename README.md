# Adana Hava Durumu Uygulaması

Adana'nın anlık hava durumunu ve 3 günlük tahminini gösteren Next.js uygulaması.

## Vercel'e Deploy

1. Bu projeyi GitHub'a yükleyin
2. [Vercel](https://vercel.com)'e gidin
3. "New Project" → Repo seçin → "Deploy"

## Yerel Geliştirme

```bash
npm install
npm run dev
```

## Teknolojiler

- Next.js 15.1.6
- React 19
- Tailwind CSS
- Open-Meteo API (ücretsiz, API key gerektirmez)
